# Elements Remover

웹페이지에서 XPath, Class, ID를 기반으로 특정 요소를 제거하는 크롬 익스텐션입니다.

## 주요 기능
- **전체 ON/OFF**: 익스텐션 아이콘을 클릭하여 기능을 활성화하거나 비활성화할 수 있습니다.
- **요소 제거 규칙 설정**:
  - **XPath**: 특정 경로의 요소를 제거합니다. (예: `//div[@class='advertisement']`)
  - **Class**: 특정 클래스를 가진 모든 요소를 제거합니다. (예: `.some-ad-class`)
  - **ID**: 특정 ID를 가진 요소를 제거합니다. (예: `#footer-ads`)
- **실시간 감지**: 페이지가 로드된 후 동적으로 추가되는 요소도 `MutationObserver`를 통해 실시간으로 감지하여 제거합니다.

## 설치 방법 (개발자 모드)
1. 이 저장소를 로컬에 다운로드합니다.
2. 크롬 브라우저를 열고 `chrome://extensions/` 주소로 이동합니다.
3. 우측 상단의 **'개발자 모드'**를 활성화합니다.
4. **'압축해제된 확장 프로그램을 로드합니다'** 버튼을 클릭합니다.
5. 이 프로젝트 폴더를 선택하여 로드합니다.

## 사용 방법
1. 익스텐션 아이콘을 클릭하여 팝업을 엽니다.
2. **'Settings'** 버튼을 클릭하여 설정 페이지로 이동합니다.
3. 제거하고자 하는 요소의 XPath, Class, 또는 ID를 입력하고 **'Add'** 버튼을 누릅니다.
4. 규칙이 추가되면 해당 규칙에 맞는 요소가 웹페이지에서 자동으로 사라집니다.
5. 팝업에서 토글 스위치를 사용하여 기능을 언제든지 켜고 끌 수 있습니다.
